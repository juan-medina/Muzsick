## Release Notes

### New Features

- **Volume Settings** — Separate Master, Radio, and DJ volume controls in the settings window.
- **Duck Level** — Configurable radio duck depth during DJ announcements.

### Bug Fixes

- Fixed settings not persisting when changed via the settings window.
- Fixed config window showing stale values instead of current session values.
- Fixed app crash on launch when Discord app ID is not configured.

### Improvements

- Settings window height increased to avoid scrolling in the AI Provider section.
